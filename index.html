What

  • Directives are functions that get run when the dom is compiled
  • Every ng-<name> is an angular directive

Why

  • Input driven DOM manipulation
  • Building declarative reusable components

How

  • Browser parses HTML into DOM tree
  • $compile parses DOM tree to find directives
  • For each directive found: run compile function and return link function
  • $compile returns linking function that wraps all the link functions from all the directives
  • linking function called where listeners, watchers are registered
  • compile function used for DOM manipulation before it's rendered and returning a link function. Good place to put all shared methods for all instances of the directive
  • link function used to register all listeners on a specific DOM element, and setup bindings to the page
